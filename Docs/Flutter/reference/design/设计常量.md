# 设计常量

**核心原则：语义化 (Semantic) · 4px 网格 (4px Grid) · 禁止魔法数字 (No Magic Numbers)**

> **单一真理源**: 所有设计常量定义在 `lib/core/theme/`，严禁硬编码。

## 间距系统

**规则**: 基于 **4px 网格系统**（网格基础单位，这是设计原则而非可配置 Token）。

**实现位置**: `lib/core/theme/app_spacing.dart`

```dart
class AppSpacing {
  static const double space1 = 4.0;      // 微观间距
  static const double space1_5 = 6.0;
  static const double space2 = 8.0;       // 组件内衬
  static const double space4 = 16.0;     // 标准容器
  static const double space6 = 24.0;      // 宏观布局
  static const double space8 = 32.0;
}
```

**使用**: 严禁使用魔法数字，必须引用 `AppSpacing` 常量。

## 圆角系统

**规则**: 使用 "超椭圆" (Superellipse) 视觉隐喻。

**实现位置**: `lib/core/theme/app_radius.dart`

```dart
class AppRadius {
  static const double sm = 4.0;      // 鹅卵石: 内部元素、列表项
  static const double md = 8.0;      // 标准容器: 卡片、输入框
  static const double xl = 16.0;     // 独立面板: 浮动面板、Toast
  static const double xxl = 24.0;    // 气泡: 模态框
  static const double full = 9999.0; // 胶囊: 按钮、标签
}
```

## 控制高度

**规则**: 遵循 **Fitts's Law** (费茨定律)，确保触控友好性。

**实现位置**: `lib/core/theme/app_sizes.dart`

```dart
class AppSizes {
  static const double controlHeightSm = 32.0;  // 次要操作 / 紧凑模式
  static const double controlHeightMd = 40.0;  // 标准输入/按钮 (默认)
  static const double controlHeightLg = 48.0;  // 底部主要 CTA
}
```
