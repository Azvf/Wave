# 响应式布局

**核心原则：断点管理 (Breakpoint Management) · 约束控制 (Constraint Control)**

## 断点定义

**规则**: 桌面端窗口大小多变，必须处理断点。

**实现位置**: `lib/core/theme/app_breakpoints.dart`

```dart
class AppBreakpoints {
  static const double compact = 600.0;    // 隐藏次要图标，缩小 Padding
  static const double standard = 1200.0;  // 标准布局
  // Wide: > 1200px (利用横向空间，如两列布局)
}
```

## 响应式 Widget

**实现**:
```dart
Widget build(BuildContext context) {
  final width = MediaQuery.of(context).size.width;
  
  if (width < AppBreakpoints.compact) {
    return CompactLayout();
  } else if (width < AppBreakpoints.standard) {
    return StandardLayout();
  } else {
    return WideLayout();
  }
}
```

## 约束管理

**规则**: 为所有卡片设置 `BoxConstraints(maxWidth: 800)`，防止在大宽屏上内容过度拉伸。

**实现**:
```dart
Center(
  child: ConstrainedBox(
    constraints: BoxConstraints(maxWidth: 800),
    child: YourCard(),
  ),
)
```
